CSS Float Practice

Float Left

This example demonstrates the float: left; property. The image is aligned to the left side, and the paragraph text wraps around the image. Float was widely used in older website layouts.

Float Right

This example demonstrates the float: right; property. The image is aligned to the right side, and the paragraph text wraps around the image from the left.

Float None

This example demonstrates the float: none; property. The image remains in its normal position and the text starts below the image instead of wrapping around it.
