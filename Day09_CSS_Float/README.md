# 🎨 Day 09 - CSS Float

Welcome to **Day09_CSS_Float**!

This folder contains my practice files for **CSS Float**. It is part of my CSS learning journey, where I explored how the `float` property can be used to position elements and create simple multi-column layouts.

The goal of this practice is to understand how the **CSS Float** property works and how it affects the normal document flow.

This practice helped me learn how to align elements to the left or right, control floating elements using the `clear` property, and build basic webpage layouts before modern layout techniques like Flexbox and Grid.
