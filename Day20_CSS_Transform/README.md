# 🎨 Day 20 - CSS Transform

Welcome to **Day20_CSS_Transform**!

This folder contains my practice files for **CSS Transform**. It is part of my CSS learning journey, where I explored how to create interactive and visually engaging user interfaces using CSS transform functions, transitions, hover effects, and responsive design techniques.

The goal of this practice is to understand how CSS **Transform** and **Transition** properties work together to create smooth animations and interactive UI components.

This practice helped me learn how to move, rotate, scale, and skew HTML elements while building responsive card layouts with modern hover effects.
