# 🎨 Day 10 - CSS Flexbox

Welcome to **Day10_CSS_Flexbox**!

This folder contains my practice files for **CSS Flexbox**. It is part of my CSS learning journey, where I explored the Flexbox layout model to create flexible, responsive, and modern web page layouts with less code.

---

# 📖 Overview

The goal of this practice is to understand how **CSS Flexbox** works and how it simplifies the process of aligning, distributing, and organizing elements within a container.

This practice helped me build responsive layouts efficiently without relying on older layout techniques like floats.

- Adding spacing using `gap`
- Controlling item size with `flex-grow`
