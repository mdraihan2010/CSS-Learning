# 🎨 Day 03 - CSS Fonts & Text Styling

Welcome to **Day03_CSS_Fonts_and_Text_Styling**!

This folder contains my practice files for **CSS Fonts and Text Styling**. It is part of my CSS learning journey, where I explored how to improve the appearance and readability of web pages using different font and text properties.

---

# 📖 Overview

The goal of this practice is to understand how CSS controls the appearance of text on a web page. I learned how to customize fonts, adjust spacing, align text, and apply decorative effects to create attractive and user-friendly designs.

This practice helped me build a strong foundation in typography, which is an essential part of modern web design.
