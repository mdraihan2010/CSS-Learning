# 🎨 Day 04 - CSS Box Model

Welcome to **Day04_CSS_Box_Model**!

This folder contains my practice files for **CSS Box Model**. It is part of my CSS learning journey, where I explored how every HTML element is represented as a rectangular box and how spacing, borders, and dimensions affect webpage layouts.

---

# 📖 Overview

The goal of this practice is to understand the **CSS Box Model**, one of the most fundamental concepts in CSS. I learned how content, padding, borders, and margins work together to determine the size and spacing of HTML elements.

This practice helped me build a strong foundation for creating well-structured, clean, and responsive web layouts.
