# 🎨 Day 07 - CSS Specificity

Welcome to **Day07_CSS_Specificity**!

This folder contains my practice files for **CSS Specificity**. It is part of my CSS learning journey, where I explored how browsers determine which CSS rule should be applied when multiple selectors target the same HTML element.

---

# 📖 Overview

The goal of this practice is to understand the concept of **CSS Specificity** and how the CSS cascade resolves style conflicts.

This practice helped me learn how different selectors have different priority levels and how to write clean, predictable, and maintainable CSS without relying on unnecessary overrides.
