# 🎨 Day 22 - CSS Table Styling

Welcome to **Day22_CSS_Table_Styling**!

This folder contains my practice files for **CSS Table Styling**. It is part of my CSS learning journey, where I explored how to design clean, responsive, and visually appealing HTML tables using various CSS properties.

The goal of this practice is to understand how CSS can be used to improve the appearance and readability of HTML tables.

This practice helped me learn how to style table borders, spacing, colors, alignment, alternating row colors, hover effects, and responsive table layouts to create professional-looking tables.
