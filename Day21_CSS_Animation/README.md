# 🎨 Day 21 - CSS Animation

Welcome to **Day21_CSS_Animation**!

This folder contains my practice files for **CSS Animation**. It is part of my CSS learning journey, where I explored how to create smooth, engaging, and interactive animations using CSS. Through hands-on practice, I learned how to animate elements, control animation behavior, and enhance user interfaces with modern visual effects.

---

# 📖 Overview

The goal of this practice is to understand how **CSS Animation** works by using `@keyframes` and various animation properties to create dynamic web pages.

This practice helped me learn how to build engaging UI animations, control animation timing, repetition, direction, and create interactive effects without using JavaScript.
