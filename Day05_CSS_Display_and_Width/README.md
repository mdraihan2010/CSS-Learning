# 🎨 Day 05 - CSS Display & Width

Welcome to **Day05_CSS_Display_and_Width**!

This folder contains my practice files for **CSS Display & Width**. It is part of my CSS learning journey, where I explored how different display properties affect the layout of HTML elements and how width-related properties help create flexible and responsive web designs.

---

# 📖 Overview

The goal of this practice is to understand how HTML elements behave with different `display` values and how to control their width effectively.

This practice helped me learn the differences between block, inline, and inline-block elements, as well as how to create responsive layouts using width, max-width, and min-width.
