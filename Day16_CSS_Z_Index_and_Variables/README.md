# 🎨 Day 16 - CSS Z-Index & Variables

Welcome to **Day16_CSS_Z_Index_and_Variables**!

This folder contains my practice files for **CSS Z-Index & Variables**. It is part of my CSS learning journey, where I explored how to control the stacking order of elements and create reusable, maintainable styles using CSS Variables.

# 📖 Overview

The goal of this practice is to understand how the **`z-index`** property controls the stacking order of overlapping elements and how **CSS Variables (Custom Properties)** make styles more reusable and easier to maintain.

This practice helped me build cleaner, more scalable, and modern CSS by reducing code duplication and improving theme customization.
