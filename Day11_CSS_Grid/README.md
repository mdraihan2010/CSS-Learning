# 🎨 Day 11 - CSS Grid

Welcome to **Day11_CSS_Grid**!

This folder contains my practice files for **CSS Grid**. It is part of my CSS learning journey, where I explored the CSS Grid Layout system to build modern, responsive, and well-structured web page layouts.

The goal of this practice is to understand how **CSS Grid** works and how it can be used to create powerful two-dimensional layouts with rows and columns.

This practice helped me learn how to organize webpage content efficiently and build complex layouts with clean and maintainable CSS.
