# 🎨 Day 06 - CSS Pseudo Class & Pseudo Element

Welcome to **Day06_CSS_Pseudo_Class_and_Element**!

This folder contains my practice files for **CSS Pseudo Classes & Pseudo Elements**. It is part of my CSS learning journey, where I explored how to style elements based on their state and create additional visual effects without modifying the HTML structure.

The goal of this practice is to understand how **Pseudo Classes** and **Pseudo Elements** work in CSS and how they can be used to create interactive, attractive, and user-friendly web pages.

This practice helped me improve my understanding of user interactions, element states, and advanced styling techniques.
